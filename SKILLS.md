# Engineering Skills & Contexts

This file defines specialized contexts that can be referenced via slash commands in NV-CODE.

## /frontend
- Focus: React, Tailwind CSS, Motion design.
- Preference: Semantic HTML, accessible components, micro-interactions.

## /backend
- Focus: Node.js, Express, Microservices, API design.
- Preference: Scalability, error handling, clean architecture.

## /security
- Focus: OWASP Top 10, Auth protocols, Data encryption.
- Preference: Zero-trust principles, hardened configurations.

## /performance
- Focus: Core Web Vitals, Bundle optimization, Caching strategies.
- Preference: Efficiency-first code, memoization.
